# Studybuddy

Studybuddy is a **web-based study platform** designed to help students connect, collaborate, and make learning more fun and effective.  
It provides tools and features for finding study partners, sharing resources, and managing study sessions.

## 🚀 Features

✔ Simple and intuitive UI  
✔ Search and find study partners  
✔ Create or join study groups  
✔ Responsive design (works on desktop & mobile)  
✔ HTML / CSS / JavaScript based front end *(edit if backend used)*
